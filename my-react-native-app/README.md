# My React Native App

This is a React Native application built with TypeScript. It serves as a template for managing trials and includes various screens and components for a modular design.

## Project Structure

```
my-react-native-app
├── src
│   ├── screens
│   │   ├── trial
│   │   │   └── TrialManagerScreen.tsx
│   │   └── index.ts
│   ├── components
│   │   └── index.ts
│   ├── types
│   │   └── index.ts
│   └── App.tsx
├── package.json
├── tsconfig.json
├── app.json
└── README.md
```

## Installation

To get started with this project, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd my-react-native-app
npm install
```

## Running the App

To run the application, use the following command:

```bash
npm start
```

This will start the Metro bundler. You can then run the app on an emulator or a physical device.

## Usage

The application includes a `TrialManagerScreen` component located in `src/screens/trial/TrialManagerScreen.tsx`. This component is responsible for rendering the trial management interface.
